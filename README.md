# Selenium Test 

This Project will execute selenium test on different browser


## Running Tests

To run tests, first clone repository using below command 

```bash
 git clone  https://github.com/mrfaru17/SeleniumFrameWork_Mukesh_Otwani.git
```

first move to project root directory

```bash
 cd SeleniumFrameWork_Mukesh_Otwani
```
Then Execute below command :

```bash
  mvn clean test 
```
